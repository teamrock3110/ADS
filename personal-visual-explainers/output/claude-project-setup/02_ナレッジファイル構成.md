## アップロードするファイル（3つ）

### 1. `wpa2-vs-wpa3.html`（完成HTML実例 — 最重要）

**役割**: CSS・HTML構造・デザインパターンの「お手本」。プロジェクト指示文で「このファイルと同じデザインで生成せよ」と指定しているため、最も重要なナレッジ。

**場所**: `output/wpa2-vs-wpa3.html`

**Claude が参照する内容**:
- グローバルCSS全体（Appleスタイルの配色・フォント・レイアウト）
- CSSクラス名とその使い方（`.hero`, `.section-light`, `.card-grid-3` 等）
- HERO → ざっくり一言 → スティッキーナビ → 問いカード → ASCII全体像 → 3つの構造 → 詳細 → 比較 → まとめ の構成パターン
- ASCIIアートの表示方法（`.ascii-wrap` + `<pre>`）
- インライン出典リンクの書き方（`.source-link`）
- セクション背景の交互パターン

### 2. `ascii-art-guide.md`（ASCIIアートパターン集）

**役割**: ASCIIアートの書き方・パターン5種（ネットワーク構成図 / 時系列シーケンス / 正常vs障害比較 / 設定値とフロー / 包含関係）のリファレンス。

**場所**: `.claude/skills/diagram-corp-eng/references/ascii-art-guide.md`

**Claude が参照する内容**:

- 5つのパターンの具体例
- ルール（幅、ラベル長、段階表現）

---

### 3. `source-citation.md`（一次情報引用ルール）

**役割**: URLを貼る閾値・情報のTier分類・HTMLでの実装方法の詳細リファレンス。

**場所**: `.claude/skills/diagram-corp-eng/references/source-citation.md`

**Claude が参照する内容**:
